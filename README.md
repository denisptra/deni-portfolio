# Deni Trio Saputra — Portfolio

Portfolio pribadi dengan tema Archive Brutalism. Dibangun dengan Next.js, TypeScript, dan Tailwind CSS v4.

## Tech Stack

- **Framework:** Next.js 16 (App Router) + React 19
- **Bahasa:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animasi:** Motion (Framer Motion)
- **Ikon:** Lucide React + Material Symbols

## Memulai

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
